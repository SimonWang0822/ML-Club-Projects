BPE from scratch

Byte Pair Encoding, a subword tokenization algorithm that merges frequently occurring character pairs into new "subword" units to create a compact and expressive vocabulary. 


      Tokenization examples:
      ------------------------------
      artificial      → ['ar', 't', 'i', 'f', 'i', 'c', 'i', 'a', 'l']
      intelligence    → ['in', 'te', 'l', 'li', 'g', 'en', 'c', 'e']
      programming     → ['pro', 'g', 'ra', 'm', 'm', 'ing']
      development     → ['de', 'v', 'e', 'l', 'o', 'p', 'm', 'en', 't']
      algorithm       → ['a', 'l', 'g', 'or', 'i', 'th', 'm', '']
      learning        → ['l', 'e', 'ar', 'ning']
